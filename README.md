# NyayaSim v2 — AI Moot Court Simulator

An interactive moot courtroom simulation and legal training platform for Indian law students.

## Quick Start Setup

### Prerequisites
- Node.js (v18+)
- Configuration files `backend/.env` and `frontend/.env` created from respective `.env.example` templates.

### Run Backend Server
```bash
cd backend
npm install
npm run dev
```

### Run Frontend Client
```bash
cd frontend
npm install
npm run dev
```
